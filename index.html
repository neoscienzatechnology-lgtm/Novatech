Desenvolvimento Full Stack

12 meses • Intermediário

Módulo 1: Fundamentos Web

  • HTML5 semântico e acessibilidade
  • CSS3 avançado: Flexbox, Grid e animações
  • JavaScript ES6+: sintaxe moderna e features
  • Versionamento com Git e GitHub
  • Responsive Design e Mobile First

Módulo 2: Front-end Moderno

  • React.js: componentes, hooks e context
  • Next.js: SSR, SSG e routing
  • TypeScript para type safety
  • State management com Redux e Zustand
  • Testes com Jest e React Testing Library

Módulo 3: Back-end com Node.js

  • Node.js e Express.js fundamentals
  • REST APIs e GraphQL
  • Autenticação e autorização (JWT, OAuth)
  • Integração com bancos de dados SQL e NoSQL
  • WebSockets e comunicação em tempo real

Módulo 4: Bancos de Dados

  • PostgreSQL e MySQL: queries avançadas
  • MongoDB: modelagem de dados NoSQL
  • ORMs: Prisma e Sequelize
  • Redis para caching
  • Otimização de queries e performance

Módulo 5: DevOps e Deploy

  • Docker e containerização
  • CI/CD com GitHub Actions
  • Deploy em AWS, Vercel e Heroku
  • Monitoramento e logs
  • Projeto final: aplicação completa

Data Science & AI

10 meses • Avançado

Módulo 1: Python para Data Science

  • Python fundamentals e programação orientada a objetos
  • Numpy e computação científica
  • Pandas para manipulação de dados
  • Matplotlib e Seaborn para visualização
  • Jupyter Notebooks e ambiente de desenvolvimento

Módulo 2: Estatística e Matemática

  • Estatística descritiva e inferencial
  • Probabilidade e distribuições
  • Álgebra linear aplicada
  • Cálculo e otimização
  • Testes de hipótese e A/B testing

Módulo 3: Machine Learning

  • Scikit-learn e algoritmos supervisionados
  • Aprendizado não supervisionado: clustering e dimensionalidade
  • Feature engineering e seleção de features
  • Validação de modelos e métricas
  • Ensemble methods e boosting

Módulo 4: Deep Learning e IA

  • Redes neurais artificiais
  • TensorFlow e PyTorch
  • CNNs para visão computacional
  • RNNs e LSTMs para séries temporais
  • Transfer learning e modelos pré-treinados

Módulo 5: Projetos e Deploy

  • MLOps e versionamento de modelos
  • APIs para modelos de ML
  • Deploy em cloud (AWS SageMaker, Azure ML)
  • Big Data com Spark e Hadoop
  • Projeto final: solução completa de IA

Cloud & DevOps

8 meses • Intermediário

Módulo 1: Fundamentos de Cloud

  • Conceitos de cloud computing
  • AWS: EC2, S3, RDS e serviços core
  • Azure: compute, storage e networking
  • Modelos IaaS, PaaS e SaaS
  • Segurança e compliance em cloud

Módulo 2: Containerização

  • Docker: images, containers e volumes
  • Docker Compose para aplicações multi-container
  • Registry e distribuição de images
  • Otimização de containers
  • Segurança em containers

Módulo 3: Orquestração com Kubernetes

  • Arquitetura do Kubernetes
  • Pods, deployments e services
  • ConfigMaps e Secrets
  • Scaling e auto-scaling
  • Helm para gerenciamento de pacotes

Módulo 4: CI/CD e Automação

  • Git workflows avançados
  • Jenkins, GitHub Actions e GitLab CI
  • Pipelines de integração e deployment
  • Testes automatizados em pipelines
  • Infrastructure as Code com Terraform

Módulo 5: Monitoramento e SRE

  • Prometheus e Grafana
  • Logging com ELK Stack
  • Incident management e on-call
  • Performance tuning
  • Projeto final: infraestrutura completa
